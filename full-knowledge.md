เทมเพลตกลางสำหรับรันแปลนบ้านผ่านความรู้ครบทั้ง 5 ชั้น — ผลลัพธ์เป็นรายงานเดียวที่แยกผลตามชั้น
พร้อมจุดที่ชั้นต่างๆ เห็นตรงกันหรือขัดแย้งกัน

> **วิธีใช้:** เมื่อ user บอกว่า "วิเคราะห์แปลนนี้" ให้ Claude โหลดเทมเพลตนี้ แล้วไล่ทีละชั้น
> ตาม checklist พร้อม cite หน้า wiki และไฟล์ใน `raw/` ทุกข้อสรุป

## 📥 ข้อมูลที่ต้องได้จากแปลน

ก่อนเริ่มวิเคราะห์ ขอข้อมูลเหล่านี้จาก user ถ้ายังไม่มี:
- [ ] ขนาดที่ดิน (กว้าง × ลึก, ม.) + หน้ากว้างติดถนน
- [ ] ประเภทถนนที่ติด (ซอย/ถนนสาธารณะ กี่เมตร)
- [ ] Zoning (ผังเมือง — สีอะไร, เขต/จังหวัดไหน)
- [ ] จำนวนชั้น + ความสูงรวม
- [ ] พื้นที่อาคารรวม (ตร.ม.)
- [ ] ทิศเหนือของแปลน (สำคัญมากสำหรับชั้น passive + ฮวงจุ้ย)
- [ ] งบประมาณโครงการ (ถ้าต้องประเมิน feasibility)
- [ ] ความต้องการพิเศษ (ผู้สูงอายุ, พระ, workshop, etc.)

---

## 🏛 ชั้นที่ 1 — กฎหมาย · *ห้ามพลาด*

**เป้าหมาย:** ตรวจการกระทำที่อาจขัดกฎหมาย ทำให้ขออนุญาตไม่ผ่าน

Checklist (drill ไปแต่ละหน้าใน `wiki/entities/law-*` ตาม jurisdiction):
- [ ] **ระยะร่น** จากเขตที่ดินทุกด้าน — ดู [[wiki/concepts/ระยะร่น|ระยะร่น]]
- [ ] **ระยะร่นถนน** — จากกึ่งกลางถนน/เขตทาง
- [ ] **FAR (อัตราส่วนพื้นที่อาคารรวม)** — ดู [[wiki/concepts/FAR-OSR-ข้อกำหนดผังเมือง|FAR / OSR]]
- [ ] **OSR (พื้นที่ว่าง)** — ขั้นต่ำตามผังเมือง
- [ ] **ความสูงอาคาร** — ข้อจำกัดจากผังเมือง + รัศมี airport (ถ้าเกี่ยวข้อง)
- [ ] **ขนาดห้องขั้นต่ำ** — ห้องนอน, ห้องน้ำ, ครัว ฯลฯ (กฎกระทรวงฉบับที่ 55)
- [ ] **ความสูงฝ้าขั้นต่ำ**
- [ ] **ทางหนีไฟ / บันได** — กรณีอาคาร ≥ 2 ชั้นหรือเข้าข่าย
- [ ] **พื้นที่จอดรถขั้นต่ำ** — ตามประเภทอาคาร + ผังเมืองเขต
- [ ] **ข้อบัญญัติท้องถิ่น** — ถ้าอยู่ กทม./เมืองใหญ่ ดูเพิ่ม

> **Output ของชั้นนี้:** ตาราง — ข้อกฎหมาย / ตัวเลขที่กฎหมายกำหนด / ตัวเลขในแปลน / ผ่าน-ไม่ผ่าน / reference

---

## 🔧 ชั้นที่ 2 — วิศวกรรม · *ทำได้จริงไหม*

**เป้าหมาย:** ตรวจว่าแบบทำได้จริงไหม จะมีปัญหาหน้างานไหม

- [ ] **ช่วง span** ของคานยาวเกินไปไหม — ต้องใช้คาน/เสาขนาดใหญ่กว่าปกติ?
- [ ] **ตำแหน่งเสา** ตรงกับ grid ไหม ขัดกับ layout ไหม
- [ ] **ระบบสุขาภิบาล** — ท่อน้ำทิ้งจากห้องน้ำชั้นบนต้องลงเสา/ผนังได้
- [ ] **slope ของท่อ** — ห้องน้ำห่างจาก riser มากเกินจะเสียความสูงฝ้า
- [ ] **ระบบไฟฟ้า** — ตำแหน่งหม้อแปลง/MDB, load center
- [ ] **การระบายอากาศกลไก** (ห้องน้ำในสุด, ครัว)
- [ ] **เงื่อนไขดิน/ฐานราก** — ถ้ามีข้อมูล soil test

> **Output:** รายการจุดที่ engineer น่าจะชี้ในรอบ coord ต่อไป

---

## 🎨 ชั้นที่ 3 — ออกแบบ · *อยู่สบายไหม*

**เป้าหมาย:** ประเมินคุณภาพการใช้สอย ไม่ใช่แค่กฎผ่าน

- [ ] **Circulation** — ทางเดินต่อเนื่อง? dead zone? ระยะทางจากประตูหน้าถึงห้องนอน?
- [ ] **Zoning ภายใน** — โซนส่วนตัว/สาธารณะแยกชัด? ห้องน้ำแขกใกล้โซน public?
- [ ] **Passive design** — ดู [[wiki/concepts/ออกแบบบ้านให้เย็นโดยไม่พึ่งแอร์|Passive design ร้อนชื้น]]
  - ทิศหน้าต่างหลัก (หลบตะวันตก?)
  - cross ventilation (มีช่องเข้า-ออกลมทั้งสองด้านไหม)
  - ชายคา / การกันแดด
- [ ] **Natural light** — ทุกห้องใช้งานหลักมีหน้าต่าง?
- [ ] **Function adjacency** — ครัวใกล้ dining, ห้องน้ำใกล้ห้องนอน, ฯลฯ
- [ ] **Proportion / scale** — ห้องที่ใช้ทุกวันได้พื้นที่สมเหตุ ไม่เสียสัดส่วน
- [ ] **Universal design / aging** — ถ้าเป็นบ้านมีผู้สูงอายุ: บันได, ห้องน้ำชั้นล่าง, step-free

> **Output:** คะแนน + ข้อเสนอปรับปรุง (ไม่ใช่ pass/fail แบบชั้นกฎหมาย)

---

## 🪷 ชั้นที่ 4 — วัฒนธรรมไทย · *เหมาะกับชีวิตไทยไหม*

**เป้าหมาย:** ดูว่าแปลนรองรับพฤติกรรมครอบครัวไทยไหม

- [ ] **ที่วางพระ / หิ้งพระ** — ทิศและตำแหน่งเหมาะสม (ไม่อยู่ห้องน้ำ/ห้องนอน, ไม่หันเข้าห้องน้ำ)
- [ ] **ศาลพระภูมิ / ศาลเจ้าที่** — พื้นที่ในแปลงที่ดิน (ไม่อยู่ใต้เงาบ้าน)
- [ ] **ครัวไทย vs ครัวฝรั่ง** — มีครัวไทยแยก (ทอด/ผัดกลิ่นแรง) ไหม
- [ ] **ชานบ้าน / semi-outdoor** — พื้นที่เปลี่ยนผ่านในอากาศร้อนชื้น ดู [[wiki/concepts/บ้านไทยโบราณ|บ้านไทยพื้นถิ่น]]
- [ ] **Gathering zone** — โซนรวมญาติ / โต๊ะกินข้าว 8–10 คน กรณีงานครอบครัว
- [ ] **พื้นที่ซักล้าง / ตากผ้ากลางแจ้ง** — ยังคงจำเป็นในไทย
- [ ] **รองเท้า** — จุดถอดรองเท้าชัดเจน (ธรรมเนียมไม่ใส่รองเท้าในบ้าน)

> **Output:** หมายเหตุเชิง soft — สิ่งที่ทำให้บ้าน "อยู่จริง" สบาย ไม่ใช่แค่ "ดูดี"

---

## ☯ ชั้นที่ 5 — ฮวงจุ้ย · *ตามใจเจ้าของบ้าน*

**เป้าหมาย:** แฟล็กประเด็นฮวงจุ้ยเบื้องต้น — ลูกค้าตัดสินเอง

- [ ] **ทิศบ้าน** (ประตูหน้า) — ตรงกับดวงเจ้าของบ้านไหม (ถ้ามีข้อมูลปีเกิด)
- [ ] **ประตูหน้า vs ประตูหลัง** ตรงกันแบบ "เงินทะลุ"?
- [ ] **ตำแหน่งเตา (ครัว)** — ห้ามหันเข้าประตู, ห้ามชนห้องน้ำ
- [ ] **ตำแหน่งเตียงนอน** — ไม่ชิดผนังติดห้องน้ำ, ไม่หันเท้าออกประตู, มีผนังหลังเตียง
- [ ] **บันได** — ห้ามตรงกับประตูหน้า
- [ ] **ห้องน้ำ** — ห้ามอยู่กลางบ้าน / บน-ใต้ห้องนอน / เหนือห้องพระ
- [ ] **ธาตุ** — น้ำ (ห้องน้ำ) ต้องไม่ปะทะไฟ (ครัว) โดยตรง

ดูพื้นฐานที่ [[wiki/concepts/ฮวงจุ้ยเบื้องต้น|ฮวงจุ้ยพื้นฐาน]]

> **Output:** list จุดที่ขัด + note ว่า "ลูกค้าต้องตัดสินใจระดับความสำคัญ"

---

## 🔗 บทสรุปข้ามชั้น

ปิดรายงานด้วยส่วนนี้:

1. **จุดที่ทุกชั้นเห็นตรงกัน** — strongly recommend (e.g. ห้องน้ำไม่ควรอยู่กลางบ้าน: ชั้นฮวงจุ้ย+วัฒนธรรม+กฎระบบท่อ เห็นตรงกัน)
2. **จุดที่ชั้นขัดแย้งกัน** — ต้อง trade-off (e.g. ตำแหน่งเตาที่ดีทางฮวงจุ้ย vs ใกล้ riser ท่อระบายอากาศ)
3. **TOP 3 ต้องแก้ก่อนส่งลูกค้า** — จัดลำดับตามความเสี่ยง (กฎหมาย > โครงสร้าง > ฟังก์ชัน > วัฒนธรรม > ฮวงจุ้ย)
4. **TOP 3 จุดดีของแปลน** — อย่าลืมบอกสิ่งที่แปลนทำได้ดี เพื่อให้คงไว้

**ระยะห่างขั้นต่ำ** จากผนังอาคารถึงขอบเขตที่ดิน/กึ่งกลางถนน/แหล่งน้ำสาธารณะ ตามกฎกระทรวงฉบับที่ 55
หมวด 4 (ข้อ 40–53) — เป็นเงื่อนไขด่านแรกที่ต้องผ่านก่อนเริ่มออกแบบจริง

> ✅ **ยืนยันจากต้นฉบับ** — ตัวเลขทั้งหมดในหน้านี้ถอดจาก `raw/papers/kot-krasuang-55-updated-2568.pdf`
> (ฉบับรวมแก้ไขถึง พ.ศ. 2568 จากสมาคมสถาปนิกสยาม)

## 1. ร่นจากถนนสาธารณะ (ข้อ 41)

| ประเภทอาคาร | กว้างถนน | ระยะร่น |
|-------------|----------|---------|
| **ทั่วไป (≤ 2 ชั้น, ≤ 8 ม.)** | < 6 ม. | จากกึ่งกลางถนน **≥ 3 ม.** |
| **สูง > 2 ชั้น หรือ > 8 ม. / ห้องแถว ตึกแถว พาณิชย์ / โรงงาน / สาธารณะ / ป้าย** | < 10 ม. | จากกึ่งกลางถนน **≥ 6 ม.** |
| เดียวกัน | 10–20 ม. | จากเขตถนน **≥ 1/10 ของกว้างถนน** |
| เดียวกัน | > 20 ม. | จากเขตถนน **≥ 2 ม.** |

**ตัวอย่าง:**
- บ้านเดี่ยวชั้นเดียวในซอยกว้าง 4 ม.: ร่นจากกึ่งกลางซอย 3 ม. → ผนังห่างจากแนวเขตถนน **1 ม.**
- บ้าน 3 ชั้น ในถนนกว้าง 12 ม.: ร่นจากเขตถนน **≥ 1.2 ม.** (= 12 × 0.1)
- บ้าน 3 ชั้น ในถนนกว้าง 25 ม.: ร่นจากเขตถนน **≥ 2 ม.**

## 2. ร่นจากแหล่งน้ำสาธารณะ (ข้อ 42)

| ประเภทอาคาร | กว้างแหล่งน้ำ | ร่น |
|------------|---------------|-----|
| ทั่วไป | ≤ 10 ม. (คู คลอง ลำราง) | **≥ 3 ม.** |
| ทั่วไป | > 10 ม. | **≥ 6 ม.** |
| อาคารขนาดใหญ่ริมแม่น้ำ บึง ทะเลสาบ ทะเล | — | **≥ 12 ม.** |

**ยกเว้น** (ไม่ต้องร่น): สะพาน, เขื่อน, รั้ว, ท่อระบายน้ำ, ท่าเรือ, ป้าย, อู่เรือ, คานเรือ

## 3. ร่นจากเขตที่ดินข้างเคียง (ข้อ 50) ⭐

### ผนังมีช่องเปิด (หน้าต่าง / ประตู / ช่องระบายอากาศ / ช่องแสง / ระเบียง)

| ความสูงอาคาร | ระยะห่างเขตที่ดินขั้นต่ำ |
|--------------|--------------------------|
| ≤ 9 ม. | **≥ 2 ม.** |
| 9–23 ม. | **≥ 3 ม.** |

### ผนังทึบ (ไม่มีช่องเปิด)

| ความสูงอาคาร | ระยะ |
|--------------|------|
| ≤ 15 ม. | **≥ 50 ซม.** |
| ≤ 15 ม. (ชิดเขต) | ได้ **ถ้าได้รับความยินยอมเป็นหนังสือ** จากเจ้าของที่ดินข้างเคียง |
| > 15 ม. | **ต้องได้รับความยินยอม** เจ้าของข้างเคียง + ร่น ≥ 50 ซม. |

**กันสาด / ลูกหลังคา / ชายคา:** ยื่นจากแนวเขตได้ ≤ **1.80 ม.**

## 4. ส่วนยื่นเหนือทางสาธารณะ (ข้อ 43)

- ส่วนที่ยื่น (กันสาด ระเบียง บัวฯ) ต้องสูงจากทางเท้า **≥ 3.25 ม.**
- ยื่นจากผนังอาคารได้ ≤ **50 ซม.** ถ้าติดริมทางสาธารณะ

## 5. อาคารสูง — ที่ว่างรอบอาคาร (ข้อ 51)

- อาคารสูง: ที่ว่างรอบอาคารขั้นต่ำ **≥ 1/8 ของความสูง แต่ไม่น้อยกว่า 3 ม. ทุกด้าน**
- ด้านที่มีทางเข้า: **≥ 6 ม.**
- รวมโครงสร้างสูงนอกอาคาร (ถังน้ำ ≥ 100 ลบ.ม. สูง ≥ 10 ม., เสาวิทยุ/โทรทัศน์สูง > 10 ม.)

## 6. ข้อยกเว้น — สร้างทดแทนอาคารเดิม (ข้อ 52)

ถ้าอาคารเดิมมีที่ว่าง/ร่นน้อยกว่าเงื่อนไข: ก่อสร้างทดแทนได้ **โดยไม่เพิ่ม** พื้นที่รวม / ความสูง / พื้นที่ปกคลุมดิน → ยกเว้นข้อ 2, 3, 4, 34, 36, 41

## จุดที่สถาปนิกพลาดบ่อย

- **ใส่หน้าต่างชิดเขตที่ดิน** — กลายเป็น "ผนังมีช่องเปิด" ต้องร่น ≥ 2 ม. (ไม่ใช่ 50 ซม.)
- **ลืมนับชายคา/กันสาด** — แม้ผนังร่นพอ แต่ชายคายื่นเกิน 1.80 ม. = ผิด
- **คิดว่าผังเมืองไม่เกี่ยว** — ผังเมืองอาจกำหนดระยะร่นเพิ่มเติมจากกฎกระทรวง 55
- **บ้านเก่าต่อเติม** — ข้อ 52 ให้ยกเว้นเฉพาะเงื่อนไขบางข้อ, อาคารใหม่เข้มกว่า

2 อัตราส่วนหลักจาก**ผังเมืองรวม** ที่จำกัดความหนาแน่นและพื้นที่ว่างของอาคาร — ใช้ร่วมกับ
**ระยะร่น** และ **ที่ว่าง 30% ของชั้น** (กฎกระทรวง 55) เป็นตัวกำหนดขนาดสูงสุดที่สร้างได้

> ✅ **ยืนยันจากต้นฉบับ** — ตัวเลขสำหรับ **กทม.** ถอดจาก `raw/papers/pang-muang-bma-landuse-2556.xls`
> จังหวัดอื่น: ต้อง ingest ผังเมืองรวมของจังหวัดนั้นๆ (ยังไม่มีใน raw/)

## 1. สูตรคำนวณ

### FAR — พื้นที่อาคารรวมต่อพื้นที่ดิน
```
FAR = พื้นที่อาคารรวมทุกชั้น ÷ พื้นที่ดิน
```
**พื้นที่อาคารรวมสูงสุด = FAR × พื้นที่ดิน**

### OSR — พื้นที่ว่างต่อพื้นที่อาคารรวม (%)
```
OSR = (พื้นที่ว่างในที่ดิน ÷ พื้นที่อาคารรวม) × 100%
```
**พื้นที่ว่างขั้นต่ำ = พื้นที่อาคารรวม × (OSR ÷ 100)**

## 3. ตัวอย่างคำนวณจริง

**โจทย์:** ที่ดิน 100 ตร.ว. (400 ตร.ม.) ในเขต ย.3 (เหลือง, หนาแน่นน้อย)

- **FAR 2.5** → อาคารรวมสูงสุด = 400 × 2.5 = **1,000 ตร.ม.**
- **OSR 12.5%** → ที่ว่างขั้นต่ำ = 1,000 × 12.5% = **125 ตร.ม.**
- หักที่ว่างจากพื้นที่ดิน → พื้นที่ปกคลุมดินสูงสุด = 400 − 125 = **275 ตร.ม.**
- ถ้าสร้าง footprint 275 ตร.ม. × จำนวนชั้น ≤ 1,000 ตร.ม. → **ประมาณ 3–4 ชั้น** (คำนวณจริงต้องหัก service)

เช็คซ้อนทับ [[wiki/concepts/พื้นที่ว่างรอบบ้าน|ที่ว่าง 30% กฎกระทรวง 55]]:
- ที่ว่างกฎกระทรวง 55 = 30% × พื้นที่ชั้นที่มากที่สุด = 30% × 275 = **82.5 ตร.ม.**
- OSR บังคับ **125 ตร.ม.** > 82.5 → ใช้ค่า 125 ตร.ม.

## 4. ความสัมพันธ์กับเงื่อนไขอื่น

| เงื่อนไข | จำกัดอะไร | กฎหมาย |
|---------|-----------|--------|
| **ระยะร่น** | Footprint (ขอบเขต) | กฎกระทรวง 55 ข้อ 50 + ผังเมือง |
| **FAR** | ปริมาตร (พื้นที่รวมทุกชั้น) | ผังเมือง |
| **OSR** | % พื้นที่ว่าง | ผังเมือง |
| **ที่ว่าง 30%** | % ของชั้นที่มากสุด | กฎกระทรวง 55 ข้อ 33 |
| **ความสูงเขต** | ชั้นสูงสุด | ผังเมือง (ย.1,ย.2 = 12 ม.) |

**ต้องผ่านทุกข้อพร้อมกัน** — ผ่านแค่ FAR/OSR ขออนุญาตไม่ได้

## 5. ขั้นตอนการเช็คโซนผังเมือง

1. ไปที่ **BMA Public Portal** (https://webportal.bangkok.go.th/public/) หรือ
   **DPT 3D Portal** (https://3d-cpd.bangkok.go.th) → ใส่เลขที่ดิน / ค้นหาถนน
2. ดู**สีโซน** (ย.1, ย.2, พ.3, ...) จากแผนที่ผังเมือง
3. อ่านตารางด้านบนตามโซนนั้น
4. คำนวณ FAR × ที่ดิน = **พื้นที่รวมสูงสุด**
5. คำนวณ OSR% × พื้นที่รวม = **ที่ว่างขั้นต่ำ**
6. เช็คเงื่อนไขอื่นใน [[wiki/entities/กฎกระทรวง-ฉบับที่-55]]
7. ถ้ามี **overlay zone** (ใกล้รถไฟฟ้า / เขตเมืองเก่า) → ตรวจเพิ่ม

## 6. จังหวัดอื่น

หน้านี้ครอบคลุม**กทม.**เท่านั้น — สำหรับจังหวัดอื่น:

- **นนทบุรี / ปทุมธานี / สมุทรปราการ / สมุทรสาคร / นครปฐม** — ใช้ผังเมืองรวมของแต่ละจังหวัด (ต่างจาก กทม.)
- **เชียงใหม่ / ภูเก็ต / หัวหิน / พัทยา** — ผังเมืองจังหวัด + ข้อบัญญัติท้องถิ่น
- **พื้นที่นอกเขตผังเมือง** — ใช้กฎกระทรวง 55 อย่างเดียว

## 1. พื้นที่ห้อง/หน่วย (ข้อ 19, 20)

| ประเภท | ขนาดขั้นต่ำ |
|-------|------------|
| **หน่วยในอาคารอยู่อาศัยรวม** (คอนโด/อพาร์ตเมนต์) | **≥ 20 ตร.ม.** |

> 💡 **8 ตร.ม.** คือขั้นต่ำของกฎหมาย — ถ้าต้องการ "ห้องนอนคู่" ที่อยู่สบาย แนะนำ 12–16 ตร.ม.

## 2. ระยะดิ่ง (ความสูงฝ้า) — ข้อ 22

| ประเภทห้อง | ระยะดิ่งขั้นต่ำ |

| ห้องขายสินค้า / ห้องประชุม / ห้องคนไข้รวม / คลังสินค้า / โรงครัว / ตลาด | **3.50 ม.** |

**วิธีวัด:** จากพื้นถึงพื้น (pour-to-pour). ถ้าเป็นใต้หลังคา วัดจากพื้นถึงยอดฝาหรือยอดผนัง

- พื้นที่ชั้นลอย **≤ 40%** ของพื้นที่ห้อง
- ระยะดิ่งจากพื้นชั้นลอยถึงพื้นชั้นบน ≥ **2.40 ม.**
- ระยะดิ่งจากพื้นห้องถึงพื้นชั้นลอย ≥ **2.40 ม.**

ระยะห่างระหว่าง**อาคารคนละหลังในที่ดินเดียวกัน** — ตามกฎกระทรวงฉบับที่ 55 ข้อ 48
สำคัญสำหรับโครงการที่มีหลายอาคาร (บ้าน + เรือนครัว + gym, หรือ townhouse complex, etc.)

> ✅ **ยืนยันจากต้นฉบับ** จาก `raw/papers/kot-krasuang-55-updated-2568.pdf` ข้อ 48 (ปรับปรุงโดยฉบับที่ 61 พ.ศ. 2550)

## กติกาหลัก: แยก 2 กรณี

### (1) ผนังฝั่งหันเข้าหากัน มี**ช่องเปิด** (หน้าต่าง / ประตู / ช่องแสง / ช่องระบายอากาศ / ระเบียง)

| อาคาร A สูง | อาคาร B สูง | ระยะห่างขั้นต่ำ |
|------------|------------|---------------|
| ไม่เกิน 9 ม. | ไม่เกิน 9 ม. | **≥ 4 ม.** |
| ไม่เกิน 9 ม. | เกิน 9 ม. แต่ไม่เกิน 23 ม. | ≥ 5 ม. |
| เกิน 9 ม. แต่ไม่เกิน 23 ม. | เกิน 9 ม. แต่ไม่เกิน 23 ม. | ≥ 6 ม. |

### (2) ผนังทั้งสองฝั่งเป็น**ผนังทึบ** (ไม่มีช่องเปิด)

| ขนาดอาคาร A | ขนาดอาคาร B | ระยะห่างขั้นต่ำ |
|-------------|-------------|---------------|
| ≤ 15 ม. | ≤ 9 ม. | **≥ 2 ม.** |
| ≤ 15 ม. | 9–23 ม. | ≥ 3 ม. |
| ≤ 15 ม. + ผนังทึบอีกฝั่ง | ≤ 9 ม. | ≥ 2.50 ม. |
| ≤ 15 ม. + ผนังทึบอีกฝั่ง | 9–23 ม. | ≥ 3.50 ม. |
| 15–23 ม. (ผนังทึบทั้งสอง) | 15–23 ม. | ≥ 1 ม. |

### ดาดฟ้า
ผนังส่วนของดาดฟ้าของอาคารด้านที่อยู่ใกล้กับอีกอาคารต้องทำการก่อสร้างเป็น**ผนังทึบสูงจากดาดฟ้า ≥ 1.80 ม.**

## ความหมายคำ

**"ช่องเปิด"** = หน้าต่าง, ประตู, ช่องระบายอากาศ, ช่องแสง, ระเบียง

**"ผนังทึบ"** = ผนังที่ไม่มีช่องเปิดใดๆ ในผนังนั้น

**ลอนหลังคา / กันสาด:** ยื่นจากผนังทึบได้ไม่เกิน **1.80 ม.**

## กรณีการใช้งานจริง

### บ้านเดี่ยวหลักกับเรือนครัว/เรือนแม่บ้าน
- ถ้าทั้งสองหลังสูง ≤ 9 ม. + หน้าต่างหันเข้าหากัน → **ต้องห่าง ≥ 4 ม.**
- ถ้าหลังเล็กก่อเป็นผนังทึบด้านที่ติดบ้านหลัก + บ้านหลักสูง ≤ 15 ม. สูงเรือนครัว ≤ 9 ม. → **ห่าง ≥ 2 ม.**

### ทาวน์เฮาส์ 3 หลัง เรียงในที่ดินเดียว
- ถ้าต้องการเว้นระยะ ใช้เงื่อนไขข้อ 48 → ผนังทึบระหว่างหลังเอา 2 ม.
- ถ้ามีหน้าต่างหันเข้ากัน 4 ม. (จริงๆ ถ้าเป็นบ้านแถวติดกันสนิท = ข้อ 3, 4 คนละเรื่อง)

## ความแตกต่างจาก "ระยะร่น" (ข้อ 50)

| เรื่อง | ข้อ 48 | ข้อ 50 ([[wiki/concepts/ระยะร่น]]) |
|-------|--------|--------------------------------------|
| วัดจาก | อาคารหนึ่งถึงอีกอาคาร (ในที่ดินเดียวกัน) | อาคารถึงขอบเขตที่ดิน |
| ใช้กรณี | มีหลายอาคารในที่ดิน | ทุกอาคาร (ด้านข้างเขตที่ดิน) |
| ค่าต่ำสุดในกรณีพื้นฐาน | 4 ม. (ผนังมีช่องเปิดทั้งคู่, อาคาร ≤ 9 ม.) | 2 ม. (ผนังมีช่องเปิด, อาคาร ≤ 9 ม.) |

ทั้งสองต้องผ่านพร้อมกันในโครงการที่มีหลายอาคาร

## ข้อผิดพลาดที่เจอบ่อย

- ออกแบบระยะระหว่าง 2 อาคาร ให้**ใกล้กว่า 4 ม.** แต่ใส่หน้าต่างทั้งสองฝั่ง → ผิดข้อ 48(1)
- ใส่ระเบียงหันเข้ากันระหว่างบ้านหลักกับเรือนแยก → "ระเบียง" = ช่องเปิด
- ปิดช่องเปิดเหลือหน้าต่างเล็กๆ เพื่อ "ทำผนังทึบ" → ยังเป็นช่องเปิดในกฎหมาย

"ที่ว่าง" = พื้นที่ในแปลงที่ดินที่ไม่มีสิ่งก่อสร้างปกคลุม — เงื่อนไข**สัดส่วนขั้นต่ำ** ตาม
กฎกระทรวงฉบับที่ 55 หมวด 3 (ข้อ 33–39)

> ✅ **ยืนยันจากต้นฉบับ** จาก `raw/papers/kot-krasuang-55-updated-2568.pdf`

## หลักการ: 30% / 10%

### ข้อ 33 — ที่ว่างขั้นต่ำ

| ประเภทอาคาร | ที่ว่างขั้นต่ำ |
|-------------|---------------|
| **อาคารอยู่อาศัย / อยู่อาศัยรวม** | **≥ 30%** ของพื้นที่ชั้นใดชั้นหนึ่งที่มากที่สุด |
| ห้องแถว / ตึกแถว / อาคารพาณิชย์ / โรงงาน / อาคารสาธารณะ / อื่นๆ ที่ไม่ใช่อยู่อาศัย | **≥ 10%** |
| อาคารที่ไม่ใช่อยู่อาศัยแต่มีส่วนอยู่อาศัยด้วย | ต้องมีที่ว่างตามเงื่อนไขอาคารอยู่อาศัย |

**"พื้นที่ชั้นที่มากที่สุด"** = floor ที่มีพื้นที่เยอะที่สุด (มักคือชั้นล่าง)
→ ถ้าชั้นล่าง = 200 ตร.ม. → ต้องมีที่ว่างในแปลงที่ดิน **≥ 60 ตร.ม.**

## ที่ว่างด้านเฉพาะ (ข้อ 34–39)

### ห้องแถว / ตึกแถวไม่ติดริมถนน (ข้อ 34)
- ด้านหน้า: ≥ **6 ม.**
- ด้านหลัง: ≥ **3 ม.** (บันไดหนีไฟภายนอกยื่นได้ ≤ 1.40 ม.)
- ด้านข้าง: ระหว่างคูหา ≥ **40 ม.** ต้องมีที่ว่างด้านข้างอาคาร ≥ 4 ม. ตลอดความลึก

### บ้านแถว (ข้อ 36)
- ด้านหน้า ≥ **3 ม.**, ด้านหลัง ≥ **2 ม.**
- ความยาวรวมต่อกลุ่ม ≤ 10 คูหา / 40 ม. — เลยนั้นต้องมีที่ว่างด้านข้าง ≥ **4 ม.**

### บ้านแฝด (ข้อ 37)
- ด้านหน้า ≥ **3 ม.** · ด้านหลัง ≥ **2 ม.** · ด้านข้าง ≥ **2 ม.**

### คลังสินค้า / โรงงาน (ข้อ 38–39)

| พื้นที่ | ด้านติดเขต | ด้านอื่น |
|--------|-----------|---------|
| 100–500 ตร.ม. | **≥ 6 ม.** | ≥ 3 ม. |
| 500–1,000 ตร.ม. | **≥ 10 ม.** | ≥ 5 ม. |
| > 1,000 ตร.ม. | **≥ 10 ม. ทุกด้าน** |  |
| โรงงานพื้นที่ ≥ 200–500 ตร.ม. | **≥ 3 ม.** · ≥ 2 ด้าน | — |
| โรงงาน ≥ 500 ตร.ม. ≤ 1,000 ตร.ม. | **≥ 6 ม.** | — |
| โรงงาน > 1,000 ตร.ม. | **≥ 10 ม.** ทุกด้าน |  |

## ความต่างจากเรื่อง FAR/OSR

| เรื่อง | ที่ว่าง (หมวด 3 กฎกระทรวง 55) | OSR ([[wiki/concepts/FAR-OSR-ข้อกำหนดผังเมือง]]) |
|-------|--------------------------------|------------------------------|
| กฎหมายใด | กฎกระทรวง 55 | ผังเมืองรวม |
| วัดจาก | % ของพื้นที่ชั้นที่มากที่สุดของอาคาร | % ของพื้นที่ดิน |
| ค่าบ้านเดี่ยวทั่วไป | 30% ของพื้นที่ชั้นล่าง | ขึ้นกับสีผังเมือง (5–10%) |
| เงื่อนไข | ต้องผ่านพร้อมกัน | ต้องผ่านพร้อมกัน |

> ⚠ **สำคัญ:** ทั้งสองต้องผ่าน**พร้อมกัน** — ผ่านแค่ข้อเดียวขออนุญาตไม่ได้

## ตัวอย่างคำนวณ (บ้านเดี่ยว)

ที่ดิน 200 ตร.ว. (800 ตร.ม.) · พื้นที่ชั้นล่าง 300 ตร.ม. · OSR 10% (ตามผังเมือง)

- **ที่ว่างตามกฎกระทรวง 55:** ≥ 30% × 300 = **90 ตร.ม.**
- **OSR ตามผังเมือง:** ≥ 10% × 800 = **80 ตร.ม.**
- ต้องมีที่ว่าง ≥ **90 ตร.ม.** (เอาค่าสูงกว่า)

## หมายเหตุ
- "ที่ว่าง" นับรวมที่ว่างรอบอาคาร + ที่ว่างกลางอาคาร (courtyard) ที่เปิดฟ้าได้
- ที่จอดรถมีหลังคา: อาจนับเป็นพื้นที่ปกคลุมดิน ขึ้นกับการตีความของเจ้าพนักงาน
- สระว่ายน้ำ, พื้นที่สวน: นับเป็นที่ว่าง

บันไดปกติและบันไดหนีไฟตามกฎกระทรวงฉบับที่ 55 หมวด 2 ส่วนที่ 3–4 (ข้อ 23–32)

> ✅ **ยืนยันจากต้นฉบับ** จาก `raw/papers/kot-krasuang-55-updated-2568.pdf`

## 1. บันไดอาคารอยู่อาศัย (บ้านเดี่ยว) — ข้อ 23

| Spec | ค่าขั้นต่ำ/สูงสุด |
|------|------------------|
| ความกว้างสุทธิ | **≥ 80 ซม.** |
| ช่วงหนึ่งสูง | ≤ **3 ม.** (ถ้าเกิน ต้องมีชานพัก) |
| ลูกตั้ง (riser) | ≤ **20 ซม.** |
| ลูกนอน (tread) | ≥ **22 ซม.** (หลังหักส่วนเหลื่อม) |
| ระยะดิ่ง (head clearance) | ≥ **1.90 ม.** |
| ชานพัก | ทุก 3 ม., ยาว ≥ ความกว้างบันได |

> 💡 Pattern ปลอดภัย: 17/28 ซม. (2 × 17 + 28 = 62, อยู่ในช่วง 60–65 ที่เดินสบาย)

## 2. บันไดอาคารอยู่อาศัยรวม / พาณิชย์ / สาธารณะ — ข้อ 24

| ขนาดชั้นเหนือรวม | ความกว้างสุทธิ |
|-------------------|---------------|
| ≤ 300 ตร.ม. | **≥ 1.20 ม.** |
| > 300 ตร.ม. | **≥ 1.50 ม.** |
| (ถ้า < 1.50 ม. ต้องมี **2 บันได** อย่างน้อย, แต่ละบันได ≥ 1.20 ม.) | |

### กรณีชุมนุมคน (ข้อ 24 วรรคสอง)
- ห้องประชุม / บรรยาย ≥ 500 ตร.ม. / ห้องรับประทาน/สถานบริการ ≥ 1,000 ตร.ม. / ชั้นรวม ≥ 2,000 ตร.ม.:
  - ต้องมี **2 บันได**, แต่ละบันได ≥ **1.50 ม.**, หรือ
  - 1 บันได ≥ **3 ม.**

### บันไดปกติของอาคารไม่ใช่อยู่อาศัย — ข้อกำหนดทั่วไป
- สูงเกิน 4 ม.: ชานพักทุก 4 ม., ระยะดิ่ง **≥ 2.10 ม.**
- ลูกตั้ง ≤ **18 ซม.**, ลูกนอน ≥ **25 ซม.**
- กว้างสุทธิ > 6 ม. และช่วงสูงเกิน 1 ม.: ต้องมีราวบันได**ทั้งสองข้าง**
- จมูกบันได: ต้องมีวัสดุกันลื่น
- ระยะห่างจากจุดไกลสุด → บันไดใกล้ที่สุด ≤ **40 ม.** (ข้อ 25)
- บันไดแบบ curved (90°): ลูกนอนเฉลี่ย ≥ **22 ซม.** (บ้านเดี่ยว) หรือ **25 ซม.** (อื่นๆ)

## 3. บันไดหนีไฟ — ข้อ 27–32 ⚠

### เมื่อไรต้องมี? (ข้อ 27)
อาคารใดที่**เข้าเงื่อนไขต่อไปนี้** นอกจากบันไดปกติ ต้องมีบันไดหนีไฟ ≥ 1 แห่ง:

1. อาคาร **≥ 4 ชั้น** สูง ≤ 23 ม.
2. อาคาร 3 ชั้น มีดาดฟ้าเหนือชั้น 3 พื้นที่ **> 16 ตร.ม.**

ต้องทำด้วยวัสดุทนไฟ + เข้าถึงได้โดยไม่มีสิ่งกีดขวาง

### ความชัน (ข้อ 28)
- ปกติ: ≤ **60°**
- ยกเว้น: ตึกแถว/บ้านแถวสูง ≤ 4 ชั้น ให้ชัน > 60° ได้ + ต้องมีชานพักทุกชั้น

### บันไดหนีไฟภายนอกอาคาร (ข้อ 29)
- กว้างสุทธิ ≥ **60 ซม.**
- ผนังส่วนที่บันไดพาดผ่าน: **ผนังทึบ ทนไฟ**
- ถ้าทอดไม่ถึงชั้นล่าง: มีบันไดโลหะเลื่อน/หย่อนลงมาจนถึงพื้นได้

### บันไดหนีไฟภายในอาคาร (ข้อ 30)
- กว้างสุทธิ **≥ 80 ซม.**
- ผนังรอบ: ทึบ + วัสดุถาวรทนไฟ
- ยกเว้นส่วนช่องระบายอากาศและประตูหนีไฟ
- **ช่องระบายอากาศทุกชั้น ≥ 1.4 ตร.ม.** เปิดสู่ภายนอก
- มีแสงสว่างเพียงพอตลอด 24 ชั่วโมง

### ประตูหนีไฟ (ข้อ 31)
- วัสดุ**ทนไฟ**
- กว้างสุทธิ ≥ **80 ซม.**, สูง ≥ **1.90 ม.**
- บานเปิด**ผลักออก**สู่ภายนอกเท่านั้น
- มีอุปกรณ์ปิดตนเอง (self-closing)
- ไม่มีธรณีหรือขอบกั้น
- เปิดได้สะดวกตลอดเวลา

### พื้นหน้าบันไดหนีไฟ (ข้อ 32)
- ด้านบันได: กว้าง ≥ ความกว้างบันได
- อีกด้าน: กว้าง **≥ 1.50 ม.**

## 4. ทางเดินเชื่อมระหว่างอาคาร — ข้อ 32/1 (เพิ่มโดยฉบับที่ 66 พ.ศ. 2559)

สำหรับอาคารขนาดใหญ่ / อาคารขนาดใหญ่พิเศษ / อาคารสูง:
- กว้าง **3–6 ม.**
- สูงจากพื้นดิน/ถนนใต้ทางเดินเชื่อมถึงส่วนล่างสุดของโครงสร้าง **≥ 5.50 ม.**
- อาคารที่มีทางเดินเชื่อมมากกว่า 1 แห่ง: ระหว่างทางเดิน **≥ 30 ม.**
- โครงสร้างหลัก: **ทนไฟ ≥ 2 ชั่วโมง**
- ไม่มีสิ่งกีดขวาง
- ห้ามสร้างทางเดินเชื่อมในบริเวณที่ต้องใช้ผนังทึบ

## รายการตรวจสอบสำหรับสถาปนิก

- [ ] อาคารเข้าเงื่อนไขต้องมีบันไดหนีไฟหรือไม่? (≥ 4 ชั้น? ดาดฟ้าใช้งาน > 16 ตร.ม.?)
- [ ] บันไดหลักกว้างพอตาม class ของอาคาร? (บ้าน 80 ซม. / อื่นๆ 1.20/1.50 ม.)
- [ ] ระยะจากจุดไกลสุดถึงบันไดใกล้สุด ≤ 40 ม.?
- [ ] ระยะดิ่งใต้บันได ≥ 1.90 ม. (บ้าน) / 2.10 ม. (อื่น)?
- [ ] ประตูหนีไฟเปิดผลักออก + ปิดเองได้?
- [ ] ลูกตั้ง/ลูกนอนตามประเภทอาคารที่ถูกต้อง?

โดยไม่ต้องรู้ดวงเจ้าของบ้าน** (ดวง/ปีเกิดเป็นเรื่องเฉพาะบุคคล ต้องถามเพิ่ม)

## 1. ประตูหน้า (大门)

- **ห้ามตรง** กับประตูหลัง — "เงินเข้าแล้วทะลุออก"
- **ห้ามตรง** กับบันได — "เงินไหลลงพื้น"
- **ห้ามตรง** กับห้องน้ำ — "เปิดประตูเห็นเรื่องสกปรก"
- **ห้ามตรง** กับเตาไฟ — "ประตูเป่าไฟ, ครอบครัวร้อน"
- ควรมีพื้นที่หน่วง (ไม่ใช่ห้องนั่งเล่นแบบเปิดโล่งทันที)

## 2. ครัว (厨房)

- **เตาไฟ** — พื้นฐานของฮวงจุ้ยครัว
  - ห้ามหันหน้าเข้าประตูครัว
  - ห้ามชิดผนังที่ติดห้องน้ำอีกฝั่ง (ไฟ vs น้ำปะทะ)
  - ห้ามใต้คาน (คานทับเตา = โชคถูกกด)
  - ห้ามตรงกับอ่างล้างจาน (ไฟชนน้ำ)
- ครัวไม่ควรอยู่กลางบ้าน — ไฟกลางบ้าน = "หัวใจร้อน"

## 3. ห้องน้ำ (卫生间)

- **ห้ามกลางบ้าน** — ธาตุน้ำกลาง = พลังงานหนักกลางบ้าน
- **ห้ามเหนือห้องนอน** โดยตรง (ห้องน้ำชั้นบน กดห้องนอนชั้นล่าง)
- **ห้ามเหนือ/ใต้ห้องพระ**
- **ห้ามตรงทางเข้าบ้าน**
- ประตูห้องน้ำควรปิดตลอด — กั้นพลังงานเสีย

## 4. ห้องนอน (卧房)

- **เตียง** — ประเด็นที่คนสนใจที่สุด
  - หัวเตียงชิดผนังทึบ (ไม่ใช่หน้าต่าง, ไม่ใช่ผนังห้องน้ำ)
  - ไม่หันเท้าเตียงออกประตู ("ทิศศพ")
  - ไม่ใต้คาน/เพดานต่ำ (กด)
  - เห็นประตูห้องจากเตียงได้โดยไม่ต้องนอนตรงกับประตู
- ห้องนอน master ควรอยู่ไกลประตูหน้า (privacy + ฮวงจุ้ย ตรงกัน)

## 5. บันได (楼梯)

- ห้ามตรงประตูหน้า
- ห้ามอยู่กลางบ้าน (ตัดบ้าน 2 ฝั่ง)
- เลี่ยงบันไดเวียน (พลังงานพุ่ง)

## 6. ทิศและธาตุ (คร่าวๆ)

| ทิศ | ธาตุ | เหมาะเป็น |
|-----|------|----------|
| เหนือ | น้ำ | ห้องนอน, ห้องทำงาน |
| ใต้ | ไฟ | ห้องนั่งเล่น |
| ตะวันออก | ไม้ | ห้องนอน, ครอบครัว |
| ตะวันตก | โลหะ | ห้องเด็ก, ห้องพักผ่อน |
| ตะวันออกเฉียงใต้ | ไม้ | เงินทอง — วางประตูทางเข้าหลัก |

> รายละเอียดเฉพาะดวงเจ้าของบ้าน (ชวด/ฉลู/ปีเกิด) ต้องปรึกษาซินแสจริง ไม่ใช่ใน scope นี้

## จุดที่ฮวงจุ้ยเห็นตรงกับชั้นอื่น

- "ห้องน้ำไม่อยู่กลางบ้าน" = **วิศวกรรม** (ท่อระบาย slope ยาก) + **กฎหมาย** (ระบบระบายอากาศยาก)
- "หัวเตียงติดผนังทึบ" = **ออกแบบ** (รู้สึกปลอดภัย, psychology)
- "ประตูหน้ารับลมดี" = **passive design** (cross ventilation)
- "ครัวไม่ติดห้องน้ำ" = **วิศวกรรม** (แยก riser) + **ใช้สอย** (กลิ่น)

ฮวงจุ้ยไม่ใช่ทุกข้อไร้เหตุผล — หลายข้อเป็น design wisdom ที่ห่อด้วยภาษาเชิงจิตวิญญาณ

แนวทางออกแบบที่ลดภาระ AC ด้วยสถาปัตยกรรมเอง — เหมาะกับไทยซึ่งร้อน ชื้น และมีฝน
ครึ่งปี บ้านพื้นถิ่นไทยทำเรื่องนี้ได้ดีมากมาหลายร้อยปี

## หลักการ 4 เรื่อง

### 1. หนีตะวันตก
- ทิศตะวันตกร้อนที่สุดช่วงบ่าย — หลีกเลี่ยงห้องที่อยู่นานในทิศนั้น
- วางห้องพยายามเรียงลำดับ: **ห้องน้ำ/ห้องครัว/คลัง** รับทิศตะวันตกแทน ห้องนอน/ห้องนั่งเล่น
- หน้าต่างฝั่งตะวันตก: ขนาดเล็ก + มีการกันแดด (ชายคายาว, louvre, ต้นไม้)

### 2. ให้ลมผ่านบ้านได้ทะลุ (Cross Ventilation)
- **ลมเข้า-ออกคนละด้าน** ของห้อง ลมถึงจะวิ่งผ่าน
- ช่องรับลมหลัก: **ฝั่งใต้/ตะวันตกเฉียงใต้** (ลมประจำภาคกลางไทย)
- ระยะห่างระหว่างช่องเข้า-ออก ไม่เกิน 5× ความกว้างห้อง
- หน้าต่างด้านเดียวไม่ใช่ ventilation — เป็น "แสง" เท่านั้น

### 3. การกันแดด (Shading)
- ชายคา/กันสาดยาวพอให้เงาครอบหน้าต่างในวันที่ดวงอาทิตย์สูง
- Rule of thumb ไทย: ชายคายื่น ≈ 1–1.5 ม. ป้องกันแดดและฝนสาด
- Louvre แนวนอน: กันแดดฝั่งใต้ · Louvre แนวตั้ง: กันแดดฝั่งตะวันออก/ตก
- ต้นไม้ผลัดใบฝั่งตะวันตก = ธรรมชาติทำงานแทน (ร่มฤดูร้อน, โปร่งฤดูหนาว — แต่ไทยไม่ผลัดใบเท่าไหร่)

### 4. สะสมความเย็นกลางคืน (Thermal Mass + Night Cooling)
- ผนังหนา/คอนกรีตเก็บความเย็นกลางคืน คายช้า → ดีสำหรับห้องนอน
- Night ventilation: เปิดหน้าต่างตี 3–6 ให้ลมเย็นไล่ความร้อนสะสม
- สังเกต: หลังคาทรงสูง + มีช่องระบายยอดจั่ว = ความร้อนลอยขึ้นไป

## สิ่งที่ไม่ควรทำในบ้านไทย

- กระจกใสทั้งผนังฝั่งตะวันตก (สวยแต่ร้อนนรก)
- หน้าต่างด้านเดียวของห้อง — AC ไม่มีทางไม่ใช้
- ดาดฟ้าแบนไม่มีฉนวนกันความร้อน
- บ้านปิดทึบ 360° — ต้องพึ่ง AC 24 ชม.

## จุดเชื่อมกับชั้นอื่น

- **ชั้นวัฒนธรรมไทย:** ชานบ้าน / ใต้ถุน / หน้าจั่วสูง = passive design ที่คนไทยรู้อยู่แล้ว — ดู [[wiki/concepts/บ้านไทยโบราณ|บ้านไทยพื้นถิ่น]]
- **ชั้นกฎหมาย:** ระยะร่นที่ดี = มีที่ให้ลมและต้นไม้ ระยะร่นน้อย = passive ทำยาก
- **ชั้นฮวงจุ้ย:** "ประตูหน้ารับลมดี" = cross ventilation พูดคนละภาษา เรื่องเดียวกัน
